## סקירה כללית

**NatID CRM** היא מערכת ניהול שירותי דרך מקיפה המיועדת לחברת "נתי שירותי דרך". המערכת מאפשרת:
- קליטת קריאות שירות מבוטים ומוקדנים
- הקצאת ספקים חכמה
- מעקב בזמן אמת אחר ספקים
- ניהול לקוחות וחוזים
- דוחות וניתוח ביצועים
- פורטל ספקים עצמאי

---

## טכנולוגיות

| רכיב | טכנולוגיה |
|------|-----------|
| Frontend | React 18.2, Vite 6, JavaScript |
| UI Framework | Radix UI, Tailwind CSS |
| State Management | TanStack React Query v5 |
| טפסים | React Hook Form, Zod |
| מפות | React Leaflet, Waze Integration |
| Backend/API | Base44 SDK |
| Functions | Deno TypeScript (Serverless) |
| שפה | עברית (RTL) |
| גרפים | Recharts, Chart.js |
| אנימציות | Framer Motion, Anime.js |
