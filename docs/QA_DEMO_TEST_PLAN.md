## 1. הכנת סביבה

### 1.1 הרצת נתוני דמו

```bash
# טעינת נתוני דמו דרך ה-function
# (דורש התחברות כ-Admin)
POST /functions/seedDemoData
Body: {}
```

**מה נוצר:**
- 3 לקוחות (חברת ביטוח, צי רכב, פרטי)
- 5 ספקים (גרר, מכונאי, צמיגי, פנימי, מנעולן)
- 10 קריאות (בכל הסטטוסים)
- דירוגי ספקים, מיקומי GPS, היסטוריית קריאות

```bash
npm test              # הרצת כל הטסטים (55 tests)
npm run test:watch    # מצב watch — רץ על כל שינוי
npm run test:coverage # דוח כיסוי קוד
```

### 1.3 הרצת בדיקות קוד

```bash
npm run lint          # בדיקת ESLint
npm run format:check  # בדיקת Prettier
npm run build         # בדיקת Build
```
