## ממצאים (לא חוסמים)

1. **קריאות רשת שאינן ממוקות במצב הדגמה** — במהלך הריצה נרשמו קריאות 404 ל-`/api/apps/null/...` (User/me, analytics/track, InvokeLLM). כלומר, חלק מהקוד (אנליטיקס, ווידג'טים מבוססי LLM) עוקף את שכבת הדמו ופונה ל-SDK האמיתי. לא משפיע על המשתמש בסביבת production, אך מייצר רעש שגיאות בקונסול במצב דמו/פיתוח מקומי.
2. **חלון הדרכה ("נתי הגרר") נחתך בקצה המסך** — בצילומי הדסקטופ והמובייל, בועת ההדרכה מוצגת חתוכה בצד שמאל. מומלץ לבדוק ידנית את מיקום רכיב ה-Tour בכל הרזולוציות.
3. **מפת הספקים תלויה ב-OpenStreetMap חיצוני** — אריחי המפה נטענים מ-tile.openstreetmap.org; ברשת חסומה המפה תוצג ריקה ללא הודעת שגיאה למשתמש.
