# Skill: Analytics & Data (ניתוח נתונים)

## מתי להשתמש
כשצריך לנתח נתונים, לשלוף מטריקות, או לבנות דוחות.

## יכולות

### 1. ניתוח קוד
```
"נתח את הקודבייס ותן לי סטטיסטיקות:
- כמה קומפוננטות יש?
- כמה דפים?
- כמה hooks?
- מה הקבצים הכי גדולים?
- איפה יש קוד כפול?"
```

### 2. ניתוח תלויות
```
"נתח את ה-dependencies שלנו:
- כמה חבילות יש?
- אילו חבילות לא בשימוש?
- אילו חבילות דורשות עדכון?
- מה ה-bundle size המשוער?"
```

### 3. ניתוח ארכיטקטורה
```
"צייר לי את הארכיטקטורה של המערכת:
- תלויות בין מודולים
- flow של נתונים
- תרשים קומפוננטות
ייצר HTML ויזואלי עם דיאגרמות"
```

### 4. ניתוח ביצועים
```
"נתח את ביצועי הבנייה:
- זמן build
- גודל bundle
- code splitting efficiency
- lazy loading coverage"
```

## ניתוח Base44 Functions
```
"נתח את כל הפונקציות ב-functions/:
- מה כל פונקציה עושה?
- אילו אינטגרציות חיצוניות יש?
- איפה יש טיפול בשגיאות חסר?
- אילו פונקציות אפשר לאחד?"
```

## דגשים
- Claude יכול לנתח קוד ולייצר דוחות בלי להריץ queries
- בקש ייצור HTML עם גרפים לויזואליזציה
- שמור תוצאות ניתוח ב-docs/ לשימוש עתידי
