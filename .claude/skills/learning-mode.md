# Skill: Learning Mode (מצב למידה)

## מתי להשתמש
כשרוצים להבין קוד חדש, ללמוד טכנולוגיה, או להעמיק בארכיטקטורה.

## פרומפטים ללמידה

### הסבר קוד
```
"הסבר לי את הקוד ב-[path] כאילו אני מפתח ג'וניור.
תסביר כל פונקציה, כל hook, ולמה הם בנויים ככה."
```

### מצגת ויזואלית
```
"צור מצגת HTML ויזואלית שמסבירה איך [feature] עובד.
כלול דיאגרמות flow, דוגמאות קוד, והערות."
```

### דיאגרמת ארכיטקטורה
```
"צור דיאגרמת ארכיטקטורה של [module] בפורמט HTML.
הראה את כל הקומפוננטות, ה-hooks, וה-data flow."
```

### Spaced Repetition
```
"אני רוצה ללמוד את [topic].
שאל אותי 5 שאלות על הנושא.
אחרי כל תשובה שלי, תן לי ציון ומשוב.
בסוף, תן לי סיכום של מה שאני יודע ומה צריך לחזק."
```

### Deep Dive
```
"עשה deep dive לתוך [file/module].
תסביר:
1. מה הוא עושה
2. למה הוא בנוי ככה
3. מה האלטרנטיבות
4. מה אפשר לשפר
5. מה הסיכונים"
```

## דגשים
- השתמש ב-/config כדי להגדיר output mode ל-"Explanatory"
- בקש דיאגרמות HTML - Claude מייצר שקופיות מצוינות
- שמור את תוצאות הלמידה ב-docs/ לעיון עתידי
